# Experimentation Playbook

The primary A/B test question is whether the checkout friction reduction variant improves payment completion compared with control.

Use conversion rate as the primary success metric and revenue or average order value as commercial guardrails. Segment results should be treated as diagnostic unless the segment hypothesis was pre-planned.

A statistically significant result usually has p-value below 0.05 and a confidence interval that excludes zero. A result can still be commercially useful if confidence is lower, but recommendations should be framed with caution.

When interpreting the test, explain the size of uplift, uncertainty, segment variation, and expected commercial impact. Do not claim causality for segments without adequate sample size and pre-registration.
